(Exemplo da estrutura que seguiremos)


- Este é um componente genérico, dentro de cada pasta de componente terá um index.tsx que é onde ficará o código do componente.
- Todas as pastas de componentes devem estar com o padrão de nome Pascal case.
- Todos os outros arquivos devem utilizar Camel case.
- Em uma pasta de componente pode haver outro componente caso este componente seja exclusivo do componente pai.
