(Uso da pasta schemas)


- Essa pasta vai ser usada para criarmos os nossos schemas de validação para formulários.
- Os custom hooks devem ser criados nessa estrutura, sempre começando o nome do arquivo e da função com schema


(
  OBS.: Os schemas são onde nós podemos criar as validações para os formulários, validações essas que impedem o usuário de conseguir fazer o submit no form até que ele atenda a todos os requisitos.
)
